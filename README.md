# Questions Local Plugin (local_questions)

This is a local plugin for Moodle 5.1 (compatible with 4.1+) that provides enhanced management and interaction with Moodle questions.

## Features

- Modern Hooks implementation for question events.
- Full multi-language support (English and Spanish).
- Privacy API compliance.
- Modern mustache-based rendering.

## Installation

1. Copy the `local_questions` folder to your Moodle's `local/` directory.
2. Visit the Notifications page in Moodle to trigger the installation.
